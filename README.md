# Landing Page - Task 1

This is a simple landing page created using **HTML** and **CSS** as part of the Full-Stack Web Development Task 1. The page is designed to be clean, responsive, and visually appealing.
## Features
Header: Contains a logo and navigation links (Home, Features, Contact).
Hero Section: Includes a title, description, and a call-to-action (CTA) button.
Features Section: Displays three key features of the product/service.
Footer: Contains copyright information and additional links (Privacy Policy, Terms of Service).

---

## Technologies Used
HTML5: For structuring the webpage.
CSS3: For styling and layout.
